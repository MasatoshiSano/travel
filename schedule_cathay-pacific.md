# パリ＆ローマ 9泊11日 旅行スケジュール
**出発：2026年6月3日（水）／帰着：6月14日（日）**
**手配：HIS フリープラン｜キャセイパシフィック航空（関空発・香港乗継）**

---

## フライト概要

| 区間 | 便名 | 出発 | 到着 | 所要 |
|------|------|------|------|------|
| 関空 → 香港HKG | **CX561** | 6/3 16:55 JST | 6/3 20:25 HKT | 約4h30m |
| 香港HKG → パリCDG | **CX261** | 6/4 00:25 HKT | 6/4 07:35 CET | 約13h45m |
| パリCDG → ローマFCO | **ITA Airways（AZ）** | 6/10 午前便 | 6/10 着 | 約2h10m | EUR 330.72 |
| ローマFCO → 香港HKG | **CX292** | 6/13 13:00 | 6/14 06:30 HKT | 約11h30m |
| 香港HKG → 関空 | CX（HKG発） | 6/14 午前 | 6/14 13:00頃 | 約3h |

> CX292（FCO→HKG）は**月・木・土のみ**運航。6/13（土）に設定。
> 燃油サーチャージ：往復約**36,800円**（日本発・HKG2区間分）
> ※2026年6月発券分は4月中旬〜下旬に正式発表予定。

### Day 5　6月7日（日）パリ 自由日

完全フリー。好きなペースでパリを満喫する日。

**おすすめ候補（組み合わせ自由）**

| カテゴリ | スポット |
|----------|----------|
| 美術館 | オルセー美術館（要予約）／オランジュリー美術館（モネの睡蓮） |
| 散策 | サンジェルマン・デ・プレ / サクレ・クール＆モンマルトル |
| 買い物 | ギャラリー・ラファイエット / マレ地区セレクトショップ |
| グルメ | サンテュアン蚤の市（日曜開催！）/ カフェ・ド・フロール |
| 夜 | セーヌ川クルーズ（バトー・ムッシュ）/ エッフェル塔ライトアップ |

> 日曜はマルシェや蚤の市が多く開かれる。ポルト・ドゥ・クリニャンクールの蚤の市（Marché aux Puces）が特におすすめ。

### Day 6　6月8日（月）⭐ モン・サン・ミッシェルへ

| 時間 | 内容 |
|------|------|
| 08:00 | ホテル朝食・チェックアウト（荷物はホテル預け） |
| 08:40 | パリ・モンパルナス駅 → **TGV** → レンヌ（約1h30） |
| 10:30 | レンヌ着 → バスまたはタクシーでモン・サン・ミッシェルへ（約1h） |
| 12:00 | **モン・サン・ミッシェル**着・チェックイン |
| 12:30 | 島内レストランでランチ（名物のふわふわオムレツ） |
| 14:00 | **修道院**見学（要入場券）→ 城壁沿い散策 |
| 17:00 | 潮の変化を観察（6月上旬は夕方の潮が美しい） |
| 夜 | 島内ホテルで夕食・ライトアップ鑑賞 |

> **潮汐チェック必須**：maree.info で6/8の潮位を確認。満潮時、島が海に浮かぶ幻想的な景色に。
> 月曜開催のため観光客が少なく、ゆったり見られる。

### Day 9　6月11日（木）⭐ 青の洞窟（カプリ島日帰り）

| 時間 | 内容 |
|------|------|
| 06:30 | ホテル出発 |
| 07:05 | **テルミニ駅**発 フレッチャロッサ → ナポリ着 08:20 |
| 09:00 | モロ・ベヴェレッロ埠頭 → **カプリ島**行き高速船（約50分） |
| 10:00 | カプリ島着 |
| 10:30 | 小型ボートで**青の洞窟（Grotta Azzurra）**へ（所要30〜45分） |
| 11:30 | カプリタウンでランチ（展望レストランから絶景） |
| 14:00 | カプリ散策 or ヴィラ・ジョヴィス |
| 15:30 | フェリーでナポリへ |
| 16:30 | フレッチャロッサでローマへ（1h10分） |
| 18:00 | ローマ帰着・夕食 |

> ⚠️ **青の洞窟は波の状態次第で入場不可**（閉鎖率 約30〜40%）。
> 代替プラン：**ポンペイ遺跡**（ナポリから20分）または**ポジターノ**（アマルフィ）。
> 当日朝に天気・波情報を確認してから判断。
