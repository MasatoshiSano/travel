## A. 貴重品・書類

| 持ち物 | 注意点・選び方 |
|--------|-------------|
| パスポート | **コピー2部**（1部はスーツケース、1部はスマホ写真）。原本と別保管が鉄則 |
| 海外旅行保険証書 | 紙＋スマホ保存。クレジットカード付帯の場合は条件確認（利用付帯 vs 自動付帯） |
| クレジットカード 2枚以上 | Visa + Mastercard推奨。**必ず別々の場所に保管**。1枚盗まれても残りで対応 |
| 現金（ユーロ） | €200〜300程度の小額紙幣（€5, €10, €20中心）。チップ・小規模店・市場用 |
| eSIM設定済みスマホ | 出発前にeSIM設定＆動作確認。到着直後から使えるように |
| 航空券・予約確認書 | すべてスマホ保存 + 紙印刷。電池切れ・通信障害時に紙が命綱 |

## B. スリ・防犯対策（最重要）

> ローマ・パリはプロのスリ集団が組織的に活動。「自分は大丈夫」が一番危険。
> 実際の被害例：ファスナーを開けて中身を抜き、丁寧にファスナーを閉める。気づいた時にはもう遅い。

| 持ち物 | なぜ必要か・選び方 |
|--------|----------------|
| **セキュリティポーチ（首下げ or 腰巻き）** | パスポート・カード・高額紙幣を服の下に隠す。RFID スキミング防止機能付き推奨。**旅の安心感が段違いに変わる** |
| **スマホ首掛けストラップ** | ひったくり防止の最強対策。特にローマの地下鉄、パリのメトロ内で効果絶大 |
| **ダミー財布** | 100円ショップの財布に少額の現金。万が一スリに遭っても被害最小限 |
| **南京錠（ダイヤル式）** | スーツケース施錠。ホステル利用時のロッカー用にも |
| **リュック用カラビナロック** | ジッパーをカラビナでロック。背後から開けられるのを防止 |

**貴重品分散ルール：**
```
パスポート原本  → セキュリティポーチ（服の下）
メインカード    → ズボン前ポケット
サブカード      → スーツケース内の別ポーチ
現金            → 3箇所に分散（ポーチ、前ポケット、スーツケース）
スマホ          → 首掛けストラップ or 前ポケット
```

## C. 機内快適グッズ（長距離フライト用）

> 日本→仁川/香港→ローマは合計15時間以上。機内の過ごし方が旅全体の疲労度を左右する。
> 機内湿度は約20%（砂漠並み）。対策なしだと肌・喉がボロボロに。

| 持ち物 | 効果・旅行者の声 |
|--------|---------------|
| **ノイズキャンセリングイヤホン** | 「AirPods Proで機内が別世界になった」。エンジン音を遮断し、睡眠の質が劇的に改善 |
| **ネックピロー（空気注入式）** | コンパクトに畳めて膨らませるタイプ。硬さ調整可能なものが◎ |
| **アイマスク（立体構造）** | 目に圧迫感がない立体型。100均より数百円の差で遮光性が段違い |
| **着圧ソックス（メディキュット等）** | エコノミー症候群予防。むくみが20〜30%軽減。到着時の脚の軽さが全然違う |
| **フットレスト（ハンモック型）** | 座席前に引っ掛けて足を浮かせる。脚の疲労が大幅に軽減 |
| **携帯スリッパ** | 靴を脱いでリラックス。機内＆ホテルでも使える |
| **保湿ミスト化粧水（80ml以下）** | 1〜2時間ごとにシュッと一吹き。肌の乾燥を防ぐ |
| **シートマスク 5〜10枚** | 就寝前に使用。到着時の肌コンディションが全然違う |
| **のどぬ〜る ぬれマスク** | 保湿効果が10時間持続。喉の乾燥・痛み防止 |
| **めぐりズム 蒸気でホットアイマスク** | 「機内で使うと天国」との声多数。リラックス効果抜群 |
| **S字フック 2〜3個** | 機内で手荷物袋を座席前に掛ける。手元に物を整理できて便利 |
| **足裏リフレッシュシート** | 到着前に貼る。長時間フライト後の足の疲労回復に |
| **ボディシート** | 乗り継ぎ時にリフレッシュ。シャワー代わりに |

## D. 7月の猛暑対策

> ローマ7月：35℃超の猛暑日あり。パリ7月：25〜30℃だが近年は猛暑日（35℃超）も。
> 1日の歩行距離は15〜25km（20,000歩以上）。暑さ＋歩きの二重負荷に備える。

| 持ち物 | なぜ必要か |
|--------|---------|
| **日焼け止め SPF50+ × 2〜3本** | ヨーロッパの紫外線は日本以上に強い。**現地品は日本人の肌に合わない場合あり**。日本から持参が確実 |
| **つば広帽子** | 直射日光対策。折りたたみ可能なものが◎ |
| **サングラス（UV400）** | 石畳の照り返しが強い。目の保護は必須 |
| **折りたたみ水筒** | ローマ・パリとも公共の水飲み場あり。ローマの「ナゾーネ」（街角の水飲み場）は安全で冷たい水が出る |
| **ハンディファン or 扇子** | 美術館の行列待ち、教会前の待機時に重宝 |
| **冷感タオル** | 水で濡らして首に巻く。ローマの炎天下で救われたとの声多数 |
| **塩分タブレット** | 熱中症予防。日本から持参（現地入手困難） |

## E. 服装・衣類

> **教会の服装ルール（厳格）：** バチカン・サンピエトロ大聖堂、ピサ大聖堂等ではノースリーブ・膝上スカート・ショートパンツで**入場拒否**される。
> 実例：ショートパンツの観光客がバスタオルを腰に巻くよう指示された。レギンス+ミニスカートでも拒否（太ももが見える）。

| 持ち物 | 選び方・工夫 |
|--------|----------|
| **薄手ストール / スカーフ × 2枚** | **最重要の万能アイテム**。教会で肩を覆う＋朝晩の冷え対策＋日差しよけ＋飛行機ブランケット代わり。1枚で4役 |
| **薄手カーディガン / ジャケット × 2〜3枚** | 朝晩は15〜20℃まで冷える。「寒暖差で体調崩した」という失敗談多数 |
| **速乾パンツ × 2〜3本** | ホテルで手洗い→一晩で乾く素材。ユニクロの感動パンツ等。ポケット多めが防犯にも◎ |
| **膝が隠れるボトムス × 1本** | 教会訪問日用。7分丈以上で確実にOK |
| **Tシャツ / トップス × 4〜5枚** | 速乾素材推奨。毎日洗って3〜4枚でローテーション可能 |
| **下着 × 5〜6日分** | 速乾素材なら手洗いローテで減らせる |
| **高機能ウォーキングシューズ** | **最も重要な持ち物の1つ**。1日2万歩の石畳歩き。HOKA・New Balance等のクッション性が高いもの。**必ず事前に履き慣らす** |
| **サンダル 1足** | ホテル室内用＆近所の外出用。ビルケンシュトック等歩けるもの |

## F. 硬水対策・スキンケア

> ヨーロッパは**硬水**（カルシウム・マグネシウム含有量が日本の5〜10倍）。
> 「シャンプー後に髪がギシギシ」「肌が突然カサカサ」は硬水が原因。現地化粧品は日本人の肌に合わないケースが多い。

| 持ち物 | なぜ日本から持参すべきか |
|--------|-------------------|
| **シャンプー＆コンディショナー（トラベルサイズ）** | 硬水で泡立ちが悪くなる。日本製の慣れたものが安心 |
| **化粧水・乳液・保湿クリーム** | 機内の乾燥＋硬水のダブルパンチ対策。ヨーロッパ製品は日本人の肌に合わないことが多い |
| **フェイスマスク × 7〜10枚** | 毎晩のケアで肌トラブル回避。「持って行って本当に良かった」の声が最も多いアイテムの1つ |
| **リップクリーム × 2〜3本** | 乾燥でカサカサになりやすい。紛失にも備えて複数本 |
| **ボディクリーム** | 硬水シャワー後の保湿に必須 |

## G. 「持って行って特別な思い出になった」アイテム

> 定番品リストにはない、旅行者の実体験から生まれた「差がつく」アイテム。

| 持ち物 | どんな体験ができるか |
|--------|----------------|
| **コンパクト双眼鏡** | 教会の天井画・フレスコ画の細部が肉眼では見えない。システィーナ礼拝堂のミケランジェロ天井画、パンテオンの天窓を双眼鏡で見た人は「全く違う体験になった」と絶賛 |
| **ワインオープナー（ソムリエナイフ）** | イタリア・フランスのスーパーで€3〜5のワインを買い、セーヌ川沿いやローマの公園でピクニック。**レストランより安く、最高の思い出になった**との声多数。※機内持ち込み不可、受託荷物に入れること |
| **軽量レジャーシート** | 上記ピクニックに必須。エッフェル塔が見えるシャン・ド・マルス公園、ローマのボルゲーゼ公園で現地の人のように過ごせる |
| **折りたたみエコバッグ × 2〜3枚** | ヨーロッパはレジ袋有料（€0.10〜0.50）。マルシェ（市場）での買い物、お土産の持ち運びに毎日使う |
| **日本のお菓子（小分け）** | 現地で出会った人へのプレゼントに最強。ホテルスタッフやツアーガイドに渡すと特別な対応をしてもらえたという体験談多数。抹茶キットカット、柿の種が人気 |
| **日本茶ティーバッグ** | ヨーロッパの緑茶は全て甘い。「旅先で飲む無糖の日本茶に癒された」という声が非常に多い。ホテルの湯沸かしで毎晩のリラックスタイム |
| **防水スマホケース** | 突然の雨、噴水近くでの撮影、水辺のピクニックで安心。実際にトレビの泉で水しぶきでスマホが濡れた体験談あり |
| **ミニ三脚 / スマホスタンド** | 2人旅なら自撮り棒より三脚。タイマー撮影で自然な写真が撮れる。「観光客に頼まなくていいので気楽」 |
| **小さなノート＋ペン** | レストランで注文を書いて見せる、道を聞く時にメモを渡す。スマホ翻訳が使えない場面で意外と活躍。旅の日記として帰国後の宝物にもなる |
| **速乾タオル（マイクロファイバー）** | ヨーロッパのホテルタオルは固い場合あり。ピクニック時、突然の雨、汗拭きなど1枚あると万能 |

## J. パッキングのコツ

**スーツケース選び**
- 7〜10日間の旅行 → **65〜75Lのハードケース（4輪キャスター）** 推奨
- ヨーロッパは石畳・階段が多い。90L超は持ち運びが地獄
- **TSAロック付き**が安心

**パッキングテクニック**
```
【スーツケース内の配置】
底部 ：靴、重い物（洗面ポーチ等）
中部 ：圧縮ポーチに入れた衣類
上部 ：すぐ取り出す物（パジャマ、充電器等）

【圧縮ポーチの分け方】
ポーチA：トップス類
ポーチB：ボトムス類
ポーチC：下着・靴下
ポーチD：スキンケア・化粧品
```

**街歩き用サブバッグ**
- **斜め掛けボディバッグ or ショルダーバッグ** が最適（リュックは背後が見えずスリに狙われやすい）
- ファスナー付き・切り裂き防止素材・ダブルジッパー
- 推奨：GREGORY、Montbell、The North Face

## K. 実は不要だったもの

> 「持って行ったけど使わなかった」という声が多いもの。荷物を減らすヒントに。

| 不要だったもの | 理由 |
|-------------|------|
| 大量の日本食インスタント | ヨーロッパのスーパーでアジア食品が安く買える |
| ドライヤー | ほぼ全てのホテルに備え付けあり |
| バスタオル | ホテルにあるので不要（速乾タオル1枚で十分） |
| ガイドブック（紙） | スマホで十分。重くてかさばるだけ |
| 多額の現金 | キャッシュレス化が進んでいる。€200〜300あれば十分 |
| 携帯ドアロック | ヨーロッパのホテルドアには構造上ほぼ装着不可 |
| 変圧器 | 最近のスマホ充電器・PC充電器は100-240V対応。変換プラグだけでOK |
