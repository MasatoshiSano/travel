# パリ 6泊8日 旅行スケジュール（大韓航空プラン）
**出発：2026年7月4日（土）／帰着：7月11日（土）**
**手配：大韓航空（関空発・仁川乗継）**

---

## フライト概要

| 区間 | 便名 | 出発 | 到着 | 所要 |
|------|------|------|------|------|
| 関空KIX → 仁川ICN | **KE722** | 7/4 09:05 JST | 7/4 10:55 KST | 約1h50m |
| 仁川ICN → パリCDG | **KE901** | 7/4 12:05 KST | 7/4 18:30 CET | 約13h54m |
| パリCDG → 仁川ICN | **KE902** | 7/10 20:35 CET | 7/11 15:00頃 KST | 約11h40m |
| 仁川ICN → 関空KIX | **KE721等** | 7/11 16:00頃 KST | 7/11 18:00頃 JST | 約2h |

> 往復とも仁川空港（ICN）**ターミナル2**で乗継。大韓航空同士のため同一エリアで完結。
> 往路乗継：仁川で**約1時間10分**（同一T2・短距離のため問題なし）
> 燃油サーチャージ：別途要確認（大韓航空公式サイトで発券時に表示）

## 日程詳細

### Day 1　7月4日（土）関空出発 → パリ到着

| 時間 | 内容 |
|------|------|
| 06:00 | 関空集合・チェックイン |
| 09:05 | **KE722** 関空KIX → 仁川ICN 出発 |
| 10:55 | 仁川ICN着・乗継（T2内・約1時間10分） |
| 12:05 | **KE901** 仁川ICN → パリCDG 出発 |
| 18:30 | パリCDG着（夏時間・中欧時間） |
| 19:30 | RER B線でパリ市内へ（約35分） |
| 20:30 | ホテルチェックイン |

### Day 2　7月5日（日）⭐ ヴェルサイユ宮殿（噴水ショーデー）

| 時間 | 内容 |
|------|------|
| 07:30 | ホテル朝食 |
| 08:10 | RER C線 → ヴェルサイユ＝リヴ・ゴーシュ駅（約40分） |
| 09:00 | **ヴェルサイユ宮殿**入場（開館直後・鏡の間・礼拝堂）※事前予約必須 |

### Day 3　7月6日（月）⭐ モン・サン・ミッシェルへ

| 時間 | 内容 |
|------|------|
| 07:30 | ホテル朝食・チェックアウト（荷物はホテル預け） |
| 08:10 | パリ・モンパルナス駅 → **TGV** → レンヌ（約1h30） |
| 10:00 | レンヌ着 → バスまたはタクシーでモン・サン・ミッシェルへ（約1h） |

| 14:00 | **修道院**見学（要入場券）→ 城壁沿い散策 |
| 17:00 | 潮の変化を観察（7月上旬は夕方の潮が美しい） |
| 夜 | 島内ホテルで夕食・ライトアップ鑑賞 |

> **潮汐チェック必須**：maree.info で7/6の潮位を確認。
> 月曜開催のため観光客が少なく、ゆったり見られる。

### Day 4　7月7日（火）⭐ モン・サン・ミッシェル朝景 → パリ帰着

| 時間 | 内容 |
|------|------|
| 06:00 | **早朝の島内散策**（朝もやの中の修道院・観光客ゼロの絶景） |
| 07:30 | ホテル朝食 |
| 08:30 | チェックアウト → バスでレンヌへ |
| 10:00 | レンヌ → TGV → パリ・モンパルナス（約1h30） |
| 12:00 | パリ着・ホテルチェックイン（荷物受取） |
| 午後 | 自由時間（ノートルダム大聖堂・シテ島・カフェめぐり等） |
| 夜 | **エッフェル塔ライトアップ**（毎時5分間点灯・22時以降が特に美しい） |

> 帰りのTGVはオンライン予約推奨（夏は混雑）。
> 7月のパリは21:00頃まで明るく、夕食後にライトアップを見る時間も十分ある。

### Day 6　7月9日（木）パリ 自由日

完全フリー。好きなペースでパリを満喫する日。

**おすすめ候補（組み合わせ自由）**

| カテゴリ | スポット |
|----------|----------|
| 美術館 | オルセー美術館（要予約）／オランジュリー美術館（モネの睡蓮） |
| 散策 | サンジェルマン・デ・プレ / サクレ・クール＆モンマルトル |
| 買い物 | ギャラリー・ラファイエット / マレ地区セレクトショップ |
| グルメ | カフェ・ド・フロール / ル・マレのユダヤ料理 |
| 夜 | セーヌ川クルーズ（バトー・ムッシュ）/ エッフェル塔ライトアップ |

> 木曜はどの美術館も比較的空いている穴場デー。

### Day 7　7月10日（金）パリ 最終日 → 帰国便

| 時間 | 内容 |
|------|------|
| 07:00 | ホテルチェックアウト（荷物を空港へ持参、または当日預け） |
| 07:30〜15:00 | **最後のパリ散策**（ノートルダム大聖堂・サン・ルイ島・お土産購入） |
| 15:00 | CDG空港へ（RER B線 約35分・または余裕を持ちタクシー） |
| 17:00 | CDG空港着・チェックイン・出国手続き |
| 19:30 | 空港内で最後のフランス料理・ワイン |
| **20:35** | **KE902** パリCDG → 仁川ICN 出発 |

> CDGは広大なため、早めの到着推奨（2〜2.5時間前）。
> 20:35発は夜便のため、チェックインは18:00〜19:00を目安に。

## 費用目安（1人あたり・2人1室）

| 項目 | 金額目安 |
|------|----------|
| 航空券（大韓航空・関空往復） | 約90,000〜140,000円 |

| 現地費用（食事・入場料・交通） | 約50,000〜80,000円 |
